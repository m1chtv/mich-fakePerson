# mich·fakePerson

A browser-based **fake person data generator** for dev, QA, and testing.

## Features
- Generates realistic **fake personal info**:  
  - Full Name  
  - Age / Birthdate  
  - Gender  
  - Address  
  - Email / Phone  
- Supports random generation with one click
- Local **recent results history**
- Clean, responsive UI

## Tech Stack
- HTML5  
- CSS3  
- Vanilla JavaScript

## How to Use
1. Open `index.html` in any modern browser.  
2. Click **Generate** to produce a fake person profile.  
3. Copy or use results for testing or mock data.

## Notes
- Runs entirely **client-side** — no backend, no storage external to your device  
- Designed exclusively for **development & QA**
- Not intended for real identity misuse
